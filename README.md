# Vexter IP Scanner

Vexter IP Scanner is a modern Android application that provides useful network tools in a simple and fast interface.

## Features

### 🔍 Scanner
- Scan IP addresses with customizable settings
- Real-time results and statistics
- Copy and export results

### ⚙️ Config Generator
- Generate multiple configurations from a template
- Support large IP lists
- Copy and download generated configurations

### 🌍 IP Checker
- Check one or multiple IP addresses at the same time
- Display connection status
- Display country, state/province, and city information
- Copy and export results

### 🕓 History
- Save previous results
- Search and manage history

### 🎨 Interface
- Modern dark theme
- Fast and responsive design
- English and Persian language support
