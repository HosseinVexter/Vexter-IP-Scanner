# Vexter IP Scanner

Vexter IP Scanner یک برنامه اندرویدی مدرن است که ابزارهای کاربردی شبکه را در یک رابط ساده و سریع ارائه می‌دهد.

## قابلیت‌ها

### 🔍 اسکنر
- اسکن آدرس‌های IP با تنظیمات قابل شخصی‌سازی
- نمایش زنده نتایج و آمار
- امکان کپی و دانلود نتایج

### ⚙️ سازنده کانفیگ
- ساخت چندین کانفیگ از یک قالب
- پشتیبانی از لیست‌های بزرگ IP
- امکان کپی و دانلود کانفیگ‌های ساخته‌شده

### 🌍 بررسی‌کننده IP
- بررسی یک یا چند آدرس IP به‌صورت هم‌زمان
- نمایش وضعیت اتصال
- نمایش اطلاعات کشور، استان و شهر
- امکان کپی و دانلود نتایج

### 🕓 تاریخچه
- ذخیره نتایج قبلی
- جستجو و مدیریت تاریخچه

### 🎨 رابط کاربری
- تم تیره مدرن
- طراحی سریع و واکنش‌گرا
- پشتیبانی از زبان‌های فارسی و انگلیسی
